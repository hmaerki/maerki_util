# maerki_util

Collected utilities used by maerki brothers

```bash
uv venv
source .venv/Scripts/activate
uv pip install -e .[dev]
```

or

```bash
uv run --with=git+https://github.com/hmaerki/maerki_util.git@main zulu
```

Available commands are:

* wxpyscan - removed
* wxhttpupload - removed
* httpupload
* zulu
* alpuma
